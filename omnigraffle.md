# OmniGraffle


## Canvases

每个文件有多个 Canvases 组成，每个 Canvase 又由多个 Layers 组成，每个 Layers 又有多个 Contents。

* `Alt+Cmd+1` —— Hide/Show Sidebar
* `Alt+Cmd+N` —— 创建 Canvases
* `Shift+Cmd+>` —— Zoom out
* `Shift+Cmd+<` —— Zoom in
* 


## Inspectors

* `Shift+Cmd+I` —— Hide/Show Inspectors
* `Cmd+1` —— Object
* `Cmd+2` —— Type
* `Cmd+3` —— Properties
* `Cmd+4` —— Canvas
* `Cmd+5` —— Documents
* `Cmd+6` —— Stencils
